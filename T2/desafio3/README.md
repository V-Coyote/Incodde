<h1>How to start</h1>
  <h3> First of all install the dependencies:</h3>
    <ul>
      <li>npm install</li>
      <li>npm run dev</li>
    </ul>
  <strong>or</strong>
  <ul>
    <li>
      yarn install
    </li>
    <li>
      yarn dev
    </li>
  </ul>
  
<h3>
  You can open the browser and access:
</h3>

<ul>
  <li>
    http://localhost:5000/crud.html
  </li>
  <li>
    http://localhost:5000/google.com
  </li>
</ul>
